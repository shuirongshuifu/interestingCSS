clip-path常见裁剪效果

inset

内边距裁剪（带圆角）

使用 inset() 实现带圆角的矩形裁剪

clip-path: inset(20px 10px 30px 10px round 10px);
circle

圆形裁剪

使用 circle() 实现圆形裁剪

clip-path: circle(80px at 50% 50%);
ellipse

椭圆裁剪

使用 ellipse() 实现椭圆裁剪

clip-path: ellipse(80px 50px at 50% 50%);
triangle

三角形裁剪

使用 polygon() 实现三角形裁剪

clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
hexagon

六边形裁剪

使用 polygon() 实现六边形裁剪

clip-path: polygon(25% 0%, 75% 0%, 100% 50%, 75% 100%, 25% 100%, 0% 50%);
star

五角星裁剪

使用 polygon() 实现五角星裁剪

clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%);
SVG path

SVG路径裁剪

引用SVG路径实现波浪形裁剪

clip-path: url(#wave-path);
cross

十字形裁剪

使用 polygon() 实现十字形裁剪

clip-path: polygon(40% 0%, 60% 0%, 60% 40%, 100% 40%, 100% 60%, 60% 60%, 60% 100%, 40% 100%, 40% 60%, 0% 60%, 0% 40%, 40% 40%);
相框

相框效果

使用polygon()函数创建相框效果

clip-path: polygon(0% 0%, 100% 0%, 100% 100%, 0% 100%, 0% 0%, 10% 10%, 10% 90%, 90% 90%, 90% 10%, 10% 10%);
